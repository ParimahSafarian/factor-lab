# factor-lab

Empirical asset pricing from scratch — factor premia, portfolio sorts, and systematic strategies in Python.

A hands-on exploration of quantitative finance fundamentals: reproducing classic results, building intuition through code, and visualizing what the textbooks describe in equations.

## Structure

```
notebooks/    # Jupyter notebooks — one per topic, numbered sequentially
src/          # Reusable modules (data loaders, regression helpers, plotting)
data/         # Downloaded datasets (gitignored, fetched by code)
outputs/      # Saved figures and tables
```
